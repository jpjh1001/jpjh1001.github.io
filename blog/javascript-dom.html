Describe what the DOM is and an example of how you might interact with it.

The DOM (document object mode) is a programming interface and it can be used with any programming language but is most commonly used with Javascript. It is a model which represents the HTML document as a tree structure. Simply put, the document in ('D'OM) can be seen as a file (in our case HTML or CSS files). The Objects (D'O'M) can be seen as the tags and the elements inside HTML and the model (DO'M') is the layout structure. By using DOM we can read, access, and update the contents of a document

Explain the difference between accessing data from arrays and objects.

Objects are like variables but can assign many values. It consists of properties which includes keys and values. An object variable may be assigned many values to one variable and it could be seen as containers for named values called properties. const car {brand: 'toyota', colour: 'red', model: 'supra'} The properties are written as 'key: value'. There are two ways to access a data from an object. If we wanted to display toyota on our screen, we could fetch this by either using car.brand; or car["brand"];

Arrays are also a special type of a variable as it can hold more than one value. Arrays can hold a lot of values under one variable name and these values can be accessed by referring to an index number. Each value in an array is assigned a number but it is important to note that array indexes start with 0. So 0 would be the first element and 1 would be the second element. For example, to access an element in an array of colours, const colours = [red, orange, yellow, blue) in order to access the orange element in the colours array, you would need to input let colour = colours[1];
