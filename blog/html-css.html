What is the difference between Margin, Border, and Padding?

I'd like to start off by mentioning that these are one of many jargons used in the IT field, and are some of the many ways that you could 'style' your webpage. However, to style a webpage you would need to link it to a file named HTML which stands for HyperText Markup Language. HTML is a language used to have a plain document uploaded on a webpage.

The terms margin, border and padding are used in the language of Cascading Style Sheets also known as CSS. It decorates the document created in HTML in order for the webpage to appear more aesethically for the viewers.

These two languages go hand in hand and are basically all you need to build a webpage of your own! I'd like to think of HTML as a person and CSS as clothing (including accessories, hairstyles and makeup). Just like how a person would give out different impressions by the way they dress, a webpage would also seem completely different depending on how it is styled.

Now that I have given you the most basic introduction what consists a webpage, I will now explain the roles of margin, border and padding and the differences between them.

During my explanations, I'd like you to think of an empty 2D box just like this one below.

The element 'margin' is used when we want to control the space outside of the box.

The element 'padding' is used when we want to control the inner space of the box.

The element 'border' is used when we want to control the border (black lines of the empty box above) of the box.

Margin

Margin is used when you want to create space around the element, making space outside of the border.

It is possible to individualy control all four sides surrounding a square from top, right, left and right.

Padding

Padding controls the amount of space within the box.

And just like margin, it is possible to individualy control all four sides surrounding a square from top, right, left and right.

What is the difference between Margin and Padding?

  • As shown in the image above, margin and padding perform similar yet different functions.
  • Margins are used when you want change the position of your element on a webpage or to create space between other nearby elements.
  • Padding is used to create space around the content of the element. It is generally used when you want to alter the size of an element.

What is a border?

A border is what is in between the margin and the padding. You can think of the border as being the frame of the empty box. It is possible to adjust the width, style, and the colour of a border. By adjusting these properties. The image above shows different types of borders which can be created.
